# Nylas Demo Composer

Nylas Composer Component in Vue 3 + vite project

## How to test

* Clone this repository
* Create a .env file at the root of the project (see below)
* Run `pnpm install`
* Run `pnpm dev`

### Env file

```
VITE_COMPOSER_ID="<COMPOSER_ID>"
VITE_USER_TOKEN="<USER_ACCESS_TOKEN>"
```
